# Garval Store Images

Place your images in this directory. The following images are required:

## Logo
- `logo-white.png` - White logo for dark backgrounds (header on scroll)
- `logo-green.png` - Green/colored logo for light backgrounds

## Hero & Headers
- `hero-olive-grove.jpg` - Homepage hero background (1920x1080 recommended)
- `shop-header.jpg` - Shop page header background
- `about-header.jpg` - About page header background
- `contact-header.jpg` - Contact page header background

## About Page
- `about-preview.jpg` - About section preview on homepage
- `about-grove.jpg` - Olive grove image
- `about-harvest.jpg` - Harvest/production image
- `about-family.jpg` - Family/team image
- `about-location.jpg` - Location/landscape image

## Products
- `product-placeholder.jpg` - Placeholder for products without images
- `product-500ml.jpg` - 500ml bottle image
- `product-750ml.jpg` - 750ml bottle image
- `product-1l.jpg` - 1L tin image
- `product-pack.jpg` - 3-pack image

## Certifications
- `eco-certification.png` - Ecological certification badge

## Favicon
- `favicon.ico` - Browser favicon

## Recommended Image Sizes
- Hero images: 1920x1080px
- Header images: 1920x600px
- Product images: 800x800px (square)
- Logos: SVG or PNG with transparent background
